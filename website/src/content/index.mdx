title: tinylight
description: Lightweight, unstyled lightbox primitives.
---

tinylight is a _lightweight_, unstyled lightbox primitive, built for simplicity, flexibility, and performance.

To get started install _tinylight_:

```bash title="pnpm"
pnpm install tinylight
```

## Basic Example

I'm working to build out more detailed examples, but for now, below is a basic, unstyled version of _tinylight_, with all currently stable features.

```tsx title="Lightbox"
import { useState } from "react";
import { Lightbox as LightboxPrimitive } from "tinylight";

export const Lightbox = () => {
  const [open, setOpen] = useState(false);
  const toggleOpen = () => setOpen(!open);

  return (
    <>
      <button onClick={toggleOpen}>Open</button>

      <LightboxPrimitive open={open} handleClose={toggleOpen}>
        <LightboxPrimitive.Items>
          <LightboxPrimitive.Item>
            <img src="https://source.unsplash.com/random/1920x1080" />
          </LightboxPrimitive.Item>
          <LightboxPrimitive.Item>
            <img src="https://source.unsplash.com/random/1920x1080" />
          </LightboxPrimitive.Item>
          <LightboxPrimitive.Pagination>
            {({ activeItem, itemsCount }) => (
              <span>
                {activeItem} / {itemsCount}
              </span>
            )}
          </LightboxPrimitive.Pagination>
        </LightboxPrimitive.Items>
        <LightboxPrimitive.Nav>
          {({ toPrev, toNext }) => (
            <>
              <button onClick={toPrev}>Previous</button>
              <button onClick={toNext}>Previous</button>
            </>
          )}
        </LightboxPrimitive.Nav>
      </LightboxPrimitive>
    </>
  );
};
```
